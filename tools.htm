实用的外部工具

查看程序

名称 介绍
Irfanview 快速和易用的图片查看软件，支持多种格式。
XNView 另一个强大的图片查看软件。Irfanview和XNView都可以用作Total Commander内置查看器（Lister）的查看插件。
Uniview 另一个图片查看软件，以查看器插件方式工作（共享软件）。
DBView
(134k)		 免费的DBF（数据库）查看软件，由Jens Richter编写。
DBFRead 免费的DBF查看软件，由ALXsoft编写。
Acrobat reader Adobe出品，查看PDF文件
Lister Total Commander内置查看器的独立版本。
新：32位版本现已推出！
Notetab 优秀的多文档文本编辑器，与Total Commander一样来自瑞士。

外部压缩程序

名称 介绍
Info-ZIP(ZIP-NT) 32位命令行压缩程序，兼容ZIP格式，推荐作为Total Commander压缩软件。
pkzip pkzip 2.04g 和 2.50 DOS （16 bit） 版本，用作Total Commander的外部压缩程序。Pkzip 2.5 Windows 和 NT 控制台版本目前尚不支持 （命令行参数不同）。
ARJ ARJ 压缩程序。请注意: Total Commander 不支持新的JAR格式，因为它的文件名不能被其它程序读取。可以使用multi-arc插件处理JAR文件。
LHA for Win32 LHA的32位版本，直接下载。
软件主页（主页为日语，下载链接为英语）
RAR 主页
UC2 Simtelnet上的下载页面（已停止开发）
ACE 主页

其它有用的工具

名称 介绍
AddTime
1.03		 改变多个文件的时间和日期，例如将每个文件的时间改变为原来的时间加上1小时30分钟（数码相机的内部时间错误等情况比较实用）。作者:Roman Korcek
Backup4All 强大的备份程序，创建普通的ZIP文件来备份，所以它们可以被Total Commander浏览（每个文件创建一个备份，或者创建用户自定义大小的备份文件）。像Total Commander一样，通过扩展语言文件支持多种不同的语言。
CombineCRC
1.0		 小工具，合并用Total Commander文件分割功能创建的文件。利用Total Commander创建的CRC文件来检查文件的完整性。作者: Andreas Kromke
Compare It! 非常不错的文件比较软件（如果TC内置的比较功能对您来说不够，那么值得一试）。也可以编辑比较文件。
Examdiff 另一个非常不错的文件比较软件。
GRBackPro Windows 专业的备份和拷贝软件 - 创建与ZIP兼容的备份文件（每个目录一个文件）。
HLP文件
(英语)		 旧格式的帮助文档。
Inireloc 用来设定配置文件 wincmd.ini 和 wcx_ftp.ini的放置目录。仅支持32位的Total Commander 4.53 和以后的版本。
JBCat 磁盘目录工具，与diskdir插件使用相同的文件的格式（见插件页面）。链接到Torry's Delphi 的原始网页(http://www.jbsoft.myweb.nl/) 似乎已成为死链。
Kryptel 强大的加密软件，通过右键菜单调用。新西兰出品，无须遵守美国的出口限制。
lptdos 独立的并行端口服务器DOS版。允许在不安装Windows的情况下向/从一台机器传输文件。不支持长文件名!
lst2mult
1.3		 一次打开多个文件。可以放在Total Commander的按钮栏中，点击按钮，调用特定的程序打开所有选中的文件。
2011/01/03:
MDocExec
1.0 		打开多个文件，与lst2multi类似，用特定程序打开所有选择的文件，每个文件使用一个程序。需要.Net framework 2.0。开发这个程序的主要原因是lst2multi被 Avira AntiVir报毒。
lst2str
1.20		 与lst2multi相似，但lst2multi多次调用程序，它只调用一次，并将所有的名称传递给命令行。
Microsoft Power Toys 对于Microsoft Windows 9x 是非常有用的扩展程序。特别是'Send to X'工具，也可以与Total Commander的右键菜单一起使用（仅限32位版本）。
RaidenFTPd FTP server 一个Windows下强大的FTP服务器程序，可以很好地配合Total Commander使用。
2013/03/02
Redate
1.1.2		 与Total Commander批量重命名工具的计数器[C]类似，但是用来修改文件的时间。可以使用来递增修改多个文件的时间，见截屏。
现在支持Unicode及x64
作者：ProgMan13 (v1.1.1), Roman Korcek
TCMenu
1.7 		为Total Commander的工具栏增加下拉菜单功能。使用标准的*.bar文件。作者：MGP Software Ltd.
2011/06/22：
TC Plugins manager
2.2.4 		TC PlugMan 是Total COmmander的一个外部插件管理器。可以用来：
- 查看已安装的查看（LS）/文件系统（FS）/打包（Packer）/内容（Content）插件列表；
- 增加/删除/配置插件；
- 临时禁用/启用插件；
- 运行/重启Total Commander。
作者：Alexey Torgashin
2011/03/28：
Ultra TC Editors 		一套编辑器，用来编辑主菜单，开始菜单，快捷菜单，扩展命令，历史，配置和快捷键。
USB installer 用来在U盘上完整（包括所有的插件和设置文件）安装/复制Total Commander的工具。也提供适用于U3 USB设备的版本下载。
wcicolib TC图标库示例（替换文件列表中的图标，如目录等）。使用资源编辑器创建自己的图标库。
XSubst 与DOS命令Subst类似，不过有对用户友好的图形界面。
添加XSubst到TC快捷菜单，启动XSubst，选择文件－配置－全局－注册快捷菜单。
作者：Luke Filewalker
